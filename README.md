## 📋 Описание предметной области
База данных предназначена для автоматизации записи клиентов в автосервис на услуги диагностики, шиномонтажа и слесарных работ. Система учитывает наличие запчастей на складе и запрещает создание записи, если необходимые детали отсутствуют. Реализован расчёт ежедневной выручки с детализацией по услугам и запчастям.

## 🛠 Стек технологий
- **СУБД**: MySQL 8.0+
- **Кодировка**: utf8mb4_unicode_ci
- **Графический клиент**: phpMyAdmin / MySQL Workbench
- **Система контроля версий**: Git + GitHub
- **Диаграммы**: draw.io / Lucidchart

## 📁 Структура репозитория
В корне проекта расположен файл README с описанием. Папка sql содержит основной скрипт создания структуры базы данных schema.sql и дополнительный файл constraints_test.sql для проверки ограничений целостности. Папка queries хранит отдельные файлы с аналитическими запросами: revenue_daily.sql для расчёта выручки, appointments_details.sql для детализации записей, categories_stats.sql для статистики по категориям услуг. В папке docs находится отчёт по практической работе в форматах Markdown и PDF, а также подпапка images со скриншотами выполнения запросов. Папка diagrams содержит исходный файл и экспорт ER-диаграммы.
